# Spring-mvc-service
Spring-mvc-service is an application for managing products
## Installation
Use [maven](https://maven.apache.org/) to install i.
```bash
mvn clean install
```
## Usage
Call the following controllers' endpoints:
"/home" (GET);
"/login" (GET, POST);
"/all (GET);
"/addProduct" (GET, POST);
"/register" (GET, POST);
